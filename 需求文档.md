# 基于RAG的金融制度类知识问答系统 - 需求文档

**版本**：V1.0  
**日期**：2026年3月22日  
**项目周期**：16周（2026年春季学期）  
**团队成员**：3人

---

## 一、项目概述

### 1.1 项目背景

金融制度知识（银行监管政策、证券行业规范、保险制度等）具有数据量大、更新频繁、专业性强的特点。传统关键词检索方式难以理解语义，导致查询结果精准度不足；人工查阅制度原文效率低下，难以满足金融从业者、高校学生的快速学习与工作需求。

### 1.2 项目目标

构建一套基于RAG（检索增强生成）技术的金融制度知识问答系统，实现：
- 金融制度文档的智能化解析与结构化存储
- 自然语言问答、关键词检索、条文关联查询三种检索模式
- 快速响应（≤2秒）与高准确率（≥85%）的问答体验
- 答案溯源与可信度评估，增强结果可信度

### 1.3 预期交付成果

| 成果类型 | 具体内容 |
|---------|---------|
| 可运行系统 | Web端问答系统 + Docker部署包 |
| 知识库 | 1000+条金融制度核心知识点 |
| 文档 | 开题报告、中期报告、验收报告、用户手册 |
| 代码 | 完整项目源码 + 单元测试（覆盖率≥70%） |
